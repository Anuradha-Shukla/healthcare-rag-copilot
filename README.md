# 🏥 Healthcare RAG Copilot

AI-powered Healthcare Report Intelligence System that analyzes uploaded medical PDFs and generates automated health insights, health scores, and recommendations.

---

## 🚀 Live Demo

🔗 https://healthcare-rag-copilot-o4zt6kkx4x8ws4xjxhaykd.streamlit.app/

---

## 📌 Features

✅ User Account Interface
✅ Upload Medical PDF Reports
✅ Extract Healthcare Data from Reports
✅ Detect Potential Health Conditions
✅ Generate AI Health Score
✅ Smart Recommendations Engine
✅ Download AI Generated Summary
✅ Interactive Dashboard

## ⚙ Installation

Clone repository:

git clone https://github.com/Anuradha-Shukla/healthcare-rag-copilot.git

Move into project:

cd healthcare-rag-copilot

Install dependencies:

pip install -r requirements.txt

Run application:

streamlit run app.py
